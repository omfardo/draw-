Omfardo Draw

Omfardo Draw

Browser-based painting studio for artists

🖌️
Painting Techniques
Oil paint, watercolor, acrylic, and the experimental Kaotik engine — each with its own physics and memory system.
⚡
Kaotik Engine
A custom brush engine built by the artist. Each stroke is influenced by the last 200 strokes via an AI memory warp system.
✦
Vector Export
Every pen, brush and shape stroke is recorded as SVG paths. Export your work as infinitely scalable vector art.
📱
No Install
Works in any browser. No account, no subscription, no loading screen. Open a tab and start drawing.

User Guide
Pen P
Precise, smooth line. Best for outlines and detailed work. Use [ ] to adjust size.
Brush B
Soft round brush, 1.4× wider than pen. Ideal for general painting and blending.
Pencil S
Textured sketch feel. 60% opacity, thinner line. Great for underdrawing and sketches.
Marker M
Flat square tip, 42% opacity. Semi-transparent streaks. Use for bold color areas.
Spray N
Airbrush with density bias toward center. Hold to build up coverage. Good for gradients and shadows.
Bucket U
Radial gradient paint pour. Soft atmospheric fills and background washes.
Smoothing: Drag the Smoothing slider up to reduce hand tremor. 0% = raw input, 95% = very slow elegant curves.
Oil Paint Y
Multi-bristle engine. Each stroke spawns ~7–12 offset threads. Core stroke plus organic satellite lines. Mimics real oil brush texture.
Watercolor W
Transparent wash base + wet bleeding dots + darkened edges. Stack strokes to build depth. Works best at 40–70% opacity.
Acrylic K
Opaque body with palette knife highlights. Adds light edge on one side, shadow on the other. Impasto specks for texture.
Kaotik Q
Custom engine. Applies noise, glitch, decay and AI memory warp to every stroke. Each mark is pulled toward where you drew before.
Kaotik presets: Raw (subtle) → Decay (fading) → Glitch (jumpy) → Void (maximum chaos) → Ghost (memory-heavy). Press V to apply a Ruin pass. Press X for symmetry.
Layer Basics
Each layer is an independent canvas. Layers composite top-to-bottom. The active layer is highlighted in gold. Click any layer to select it.
Operations
+ Add · − Delete · ⧉ Duplicate · ↑↓ Reorder · ↓⊕ Merge with layer below · ⊕ Flatten all layers into one
Blend Modes
Each layer has its own blend mode: Normal, Multiply, Screen, Overlay, Darken, Lighten, Difference, Luminosity. Change it in the Layers panel.
Tip: Double-click a layer name to rename it. Click the 👁 icon to hide/show a layer without deleting it.
Eraser E
Pixel-level eraser using destination-out compositing. Erases only on the active layer.
Fill G
Flood fill with tolerance. Fills contiguous area of similar color on the active layer.
Eyedropper I
Samples color from the composite view (all visible layers). Click anywhere to pick.
Text T
Click to place text. Choose font, size, bold, italic. Press Ctrl+Enter or Shift+Enter to commit. Text is rendered on the active layer.
Pan H
Drag the canvas. Also: hold Space anytime to temporarily pan. Middle mouse button also pans.
Shapes L R O
Line, Rectangle, Ellipse. Click and drag to draw. Shapes are live-previewed while dragging.
PNG
Lossless. Supports transparent background. Best for sharing and archiving.
JPG
Compressed. Adjust quality 10–100%. Smaller file size. No transparency.
WebP
Modern format. Smaller than PNG with better quality than JPG. Supported by all modern browsers.
SVG ✦ Vector
Records all pen, brush and shape strokes as scalable vector paths. Infinitely zoomable. Editable in Illustrator, Inkscape, Figma.
Layer export control: In the Export panel, check/uncheck individual layers before downloading. Useful for removing text or sketch layers.
